# BakeBay — Функциональные требования

Каталог требований, сгруппированных по актору. Описывает **что** система делает, не как.

---

## 1. Покупатель (Buyer)

### 1.1 Каталог и поиск

- FR-B01: Просматривать список товаров с пагинацией
- FR-B02: Фильтровать товары по категории, ценовому диапазону, рейтингу, диетическим ограничениям, продавцу
- FR-B03: Видеть карточку товара с главным изображением, ценой, рейтингом, продавцом
- FR-B04: Переходить на страницу детали товара

### 1.2 Детальная страница товара

- FR-B05: Просматривать галерею изображений с переключением превью
- FR-B06: Читать полное описание, условия хранения, состав, вес, размер, срок хранения
- FR-B07: Видеть диетические метки (без глютена, веганский и т.д.)
- FR-B08: Видеть карточку продавца с переходом к его товарам
- FR-B09: Читать отзывы с оценками и распределением звёзд
- FR-B10: Видеть похожие товары

### 1.3 Корзина

- FR-B11: Добавить товар в корзину с указанием количества
- FR-B12: Изменить количество позиции в корзине
- FR-B13: Удалить позицию из корзины
- FR-B14: Видеть подытог, скидку, доставку, налог и итого
- FR-B15: Применить промо-код
- FR-B16: Очистить корзину
- FR-B17: Корзина сохраняется между сессиями (localStorage)

### 1.4 Оформление заказа

- FR-B18: Оформить заказ из корзины
- FR-B19: Использовать сохранённые адрес и способ оплаты
- FR-B20: После оформления корзина очищается, пользователь получает ID заказа

### 1.5 История заказов

- FR-B21: Видеть список своих заказов с фильтром по статусу
- FR-B22: Просматривать детали заказа: позиции, ингредиенты, статус, дата, сумма
- FR-B23: Видеть таймлайн изменений статуса заказа

### 1.6 Отзывы

- FR-B24: Оставить отзыв с оценкой (1–5) и текстом на товар
- FR-B25: Видеть ответ продавца на свой отзыв

### 1.7 Профиль

- FR-B26: Просматривать и редактировать личные данные (имя, email, телефон, адрес)
- FR-B27: Сменить пароль (подтверждение текущего пароля)
- FR-B28: Управлять сохранёнными адресами доставки

---

## 2. Продавец (Seller)

### 2.1 Регистрация

- FR-S01: Заполнить форму и стать продавцом (имя магазина, описание, контакты, ИНН)
- FR-S02: Профиль продавца привязан к учётной записи пользователя

### 2.2 Управление товарами

- FR-S03: Создать товар: название, цена, себестоимость, описание, категория, условия хранения, SKU, вес, размер, срок
  хранения
- FR-S04: Загрузить изображения товара (S3); отметить главное изображение, задать порядок
- FR-S05: Привязать диетические ограничения к товару
- FR-S06: Привязать ингредиенты с количеством и единицей измерения
- FR-S07: Установить статус товара: `active` / `draft` / `hidden`
- FR-S08: Включить отслеживание остатков и уведомление о низком стоке
- FR-S09: Редактировать товар (только владелец)
- FR-S10: Удалить товар

### 2.3 Управление запасами (ингредиенты)

- FR-S11: Просматривать список ингредиентов с текущим стоком и статусом (`ok` / `low` / `out`)
- FR-S12: Дополнить склад ингредиента (restock): обновить количество, порог алерта, единицу, цену закупки
- FR-S13: Видеть сводку ингредиентов по конкретному заказу
- FR-S14: Видеть общий список закупок для выполнения всех активных заказов

### 2.4 Управление заказами

- FR-S15: Видеть список заказов, содержащих свои товары
- FR-S16: Изменить статус заказа по разрешённым переходам конечного автомата:
  `ordering → processing → payed → processed → in_progress → delivering → delivered`

### 2.5 Отзывы

- FR-S17: Просматривать отзывы покупателей на свои товары
- FR-S18: Ответить на отзыв покупателя

### 2.6 Аналитика и дашборд

- FR-S19: Видеть KPI-дашборд: выручка, количество заказов, средний чек, топ товары
- FR-S20: Видеть последние заказы и последние отзывы

### 2.7 Оптимизация (инструменты LP)

- FR-S21: Рассчитать минимальную стоимость закупки ингредиентов для выполнения всех активных заказов (
  `computeMinPurchaseCost`)
- FR-S22: Рассчитать максимально возможную выручку с учётом текущих складских ограничений (`computeMaxProfit`)
- FR-S23: Экспортировать список закупок в CSV
- FR-S24: Экспортировать смету продукта (себестоимость, маржа) в CSV
- FR-S25: Экспортировать смету заказа (ингредиенты, затраты) в CSV

---

## 3. Администратор (Admin)

- FR-A01: Просматривать список всех пользователей
- FR-A02: Изменить роль пользователя (`customer` / `seller` / `admin`)
- FR-A03: Управлять продавцами (создание, просмотр, удаление)

---

## 4. Система (System / Cross-cutting)

### 4.1 Аутентификация и сессии

- FR-SYS01: Регистрация: валидация email-формата, пароль ≥ 6 символов, имя из двух частей
- FR-SYS02: Вход по email + пароль, возврат JWT-токена
- FR-SYS03: Выход из системы (инвалидация сессии на клиенте)
- FR-SYS04: Верификация JWT при каждом защищённом запросе
- FR-SYS05: Сессия хранит `userId` и `role`

### 4.2 Файловое хранилище

- FR-SYS06: Загрузка изображений товара в S3 / MinIO
- FR-SYS07: Удаление изображений при обновлении или удалении товара

### 4.3 Уведомления

- FR-SYS08: Toast-уведомления (info / success / warning / error) с автодисмиссом
- FR-SYS09: Максимум 5 одновременных уведомлений

### 4.4 Категории и справочники

- FR-SYS10: Справочник категорий товаров
- FR-SYS11: Справочник диетических ограничений
